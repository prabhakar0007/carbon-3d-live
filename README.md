# 🌍 Carbon 3D Tracker

A modern web application for tracking personal carbon footprint and comparing it with global emissions data.

## Features

- 📊 **Carbon Footprint Calculator** - Track emissions from various activities
- 🌍 **Global Comparison** - Compare your emissions with global averages
- 📈 **Weekly Trends** - Visualize your carbon footprint over time
- 🎯 **Goal Setting** - Set and monitor personal carbon reduction goals
- 🏆 **Leaderboard** - Compete with others to reduce emissions
- 💡 **Personalized Insights** - Get actionable recommendations
- 📱 **Responsive Design** - Works on desktop and mobile devices

## Tracked Activities

- 🚗 Car Travel (km)
- 🚌 Bus Travel (km)
- ✈️ Flights (hours)
- 💡 Electricity Usage (kWh)
- 🍖 Non-vegetarian Meals

## Tech Stack

- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **3D**: Three.js + React Three Fiber
- **Testing**: Vitest + React Testing Library

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit `http://localhost:5173` to see the app.

## Build

```bash
npm run build
```

## Testing

```bash
npm test
```

## Environment Variables

Create a `.env` file in the root directory:

```
VITE_API_BASE_URL=http://localhost:3000
```

## Deployment

### Vercel

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Vercel will automatically deploy on every push

### Manual Deployment

```bash
npm run build
# Deploy the 'dist' folder to your hosting service
```
